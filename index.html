Tentang

Mendesain sebuah interface merupakan suatu kegiatan yang saya biasa lakukan, dengan skill yang saya pelajari baik di kampus maupun otodidak membuat saya melangkah lebih baik dalam dunia desain digital.

Beberapa pengalaman yang sudah saya rasakan adalah ketika mendesain banyak web client pada platform freelance sehingga saya menerima banyak masukan positif terkait skill saya dalam mendesain sebuah user interface.

Pada lingkungan kampus saya memiliki beberapa proyek dalam desain web ataupun desain lainnya seperti videgrafi dan lain sebagainya. Pada program ini saya berkolaborasi dengan beberapa individu baik rekan mahasiswa dan mahasiswi maupun para bapak dosen dan ibu dosen sehingga proyek akan lebih fleksible dan lebih cepat untuk di deploy.
